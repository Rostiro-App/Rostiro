Nothing here is stale-but-still-linked; if any of these were still load-bearing, they'd still be in the repo root (see the note on `Rostiro_Marketing_System_v1.md`, which stayed there for exactly that reason).

1. `deep-research-report (1).md` — raw competitive/behavioral research (Gemini deep research output), the earliest input.
2. `Rostiro_Marketing_Brief.md` — the founder's own detailed brief, written from that research.
3. `Rostiro_Marketing_Plan_v1.md` — the first structured plan built from the brief, audited against the PRD and brand kit.

That plan became `Rostiro_Marketing_System_v1.md` (still in the repo root, not archived, because `Rostiro_Marketing_System_v2.md` — the current strategy doc — still actively references several of its tactical sections: Reddit templates, Discord structure, X/TikTok script banks). `v2` is what's actually current; see `../../Rostiro_Marketing_Docs_INDEX.md` for the full current doc map.
